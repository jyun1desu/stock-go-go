### 使用技術
- Vue CLI
- Vuex
- Vue Router
- Sass
- vue-chart.js
--
### 實現功能
- 個股資料選單，可以在個股資料間快速切換 
- 以股名或股號搜尋個股資料 
- 可以分享網址
- 首欄凍結
- 呈現符合使用者習慣
    - 數字至右對齊
    - 縮排
    - 負值呈現
    - 數值取到小數點第二位
- 搜尋功能
  - 查無股名／股號相關資料提示
  - 即時顯示和關鍵字相關股名／股號
